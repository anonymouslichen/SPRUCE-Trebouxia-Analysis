This directory is for outputs generated by the analysis scripts.

- `evernia-s22-primer-trimmed-single-end.qzv` - S22 quality plots

**Exported Data**:
- `spruce-evernia-feature-table.tsv` - OTU table in tab-separated format (for R)

### From R Analysis (`02_Trebouxia_analysis.R`)

**Figures** (publication-quality):
- `Figure_8A_community_composition.png/pdf` - Stacked bar plot of OTU composition
- `Figure_8A_phylogram.png/pdf` - Phylogenetic tree legend
- `Figure_8B_clade_I_temperature.png/pdf` - Regression plot

**Statistical Outputs** (printed to console):
- Linear regression results: Clade I proportion vs. temperature
- Wilcoxon test results: Core vs. tips diversity comparison
- Median Shannon diversity values

## File Organization

We recommend organizing your results as:

```
results/
├── qiime2_outputs/
│   ├── tables/           # .qza files
│   ├── visualizations/   # .qzv files
│   └── exported/         # .tsv files
├── figures/
│   ├── Figure_8A_bars.pdf
│   ├── Figure_8A_tree.pdf
│   └── Figure_8B_regression.pdf
└── statistics/
    └── statistical_results.txt  # Save console output
```

## Saving Figures from R

To save figures when running the R script, add these commands after each plot:

```r
# For ggplot objects
ggsave("results/figures/Figure_8A_bars.pdf", 
       width = 10, height = 6, units = "in")

# For ggtree objects
ggsave("results/figures/Figure_8A_tree.pdf", 
       plot = t,  # or whatever you named the tree plot
       width = 4, height = 6, units = "in")
```

## Comparing to Published Results

Your results should match the published figures in:
- Meyer et al. 2023, American Journal of Botany 110(2): e16114
- Figure 8A (panels showing OTU composition and phylogram)
- Figure 8B (clade I proportion regression)

**Expected statistical results**:
- Clade I vs. temperature: p = 0.0011, R² = 0.159, slope = 0.0196
- Core vs. tips diversity: p = 0.0151
- Median diversity (core): 0.3337
- Median diversity (tips): 0.1253

## Troubleshooting

**If results don't match**:
1. Verify QIIME2 version (2021.4)
2. Check filtering thresholds were applied correctly
3. Ensure rarefaction depth is 1790
4. Confirm unknown OTU renaming was done correctly
5. Check that duplicate OTUs were aggregated by first 3 characters

**If visualizations look different**:
1. Verify factor levels in plots match the script
2. Check color codes match exactly
3. Ensure ggplot2 version is current

## Notes

- `.qza` and `.qzv` files are binary QIIME2 formats
- These files can be large (100s of MB) and should not be committed to GitHub
- Keep only final publication figures and summary statistics in version control
- Document any deviations from expected results
